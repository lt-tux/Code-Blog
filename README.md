# Code-Blog

A personal blog website for documenting my coding journey, sharing code examples, and building a community of developers.

## Features

- Multi-page responsive website
- Blog posts with filtering by category
- Code examples with syntax highlighting
- Contribution system for community submissions
- About page with skills and coding philosophy
- Mobile-friendly design

## Pages

1. **Home** - Landing page with hero section, stats, and recent blog posts
2. **Blogs** - All blog posts with filtering functionality
3. **Examples** - Code snippets with copy functionality
4. **Contribute** - Form for community contributions
5. **About** - Personal information and coding philosophy
